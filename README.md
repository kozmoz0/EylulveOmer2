# Eylül ❤️

Bazı anılar hiç unutulmaz. Bu proje, kalbinde selameti olan ve geçmiş duygularını sanat haline dönüştüren bir kişinin, Eylül'e adanmış duygusal bir web sitesidir.

## 📝 Proje Hakkında

"Seni hiçbir zaman unutamadım Eylül" sözleriyle kurulan bu site, paylaşılan anıları ve duyguları modern web teknolojileriyle sunmaktadır. 

Sayfada yer alan özellikler:
- 💝 Duygusal giriş bölümü
- 🖼️ İnteraktif fotoğraf galerisi
- ⏱️ Başlangıç tarihinden itibaren geçen süreyi gösteren sayaç
- 💌 Kalp ikon animasyonları
- ✨ Responsive ve modern tasarım

## 🌐 Siteyi Ziyaret Edin

👉 **[Sayfayı Aç](https://kozmoz0.github.io/EylulveOmer2/)**

## 💻 Teknolojiler

- HTML5
- CSS3 (Responsive Design, Animations)
- Vanilla JavaScript

## 📸 İçerik

Site, paylaşılan anı fotoğrafları ve duyguların ifadesi olarak yazılmış şiirvari metinlerle oluşturulmuştur.
